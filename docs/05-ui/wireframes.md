# 와이어프레임

## 1. 개요

각 화면의 상세 와이어프레임과 컴포넌트 명세를 정의한다.

---

## 2. 공통 컴포넌트

### 2.1 Header

```
┌─────────────────────────────────────────────────────────┐
│  🎯 VOC 자동 처리       [VOC 입력]  [Ticket 목록]       │
└─────────────────────────────────────────────────────────┘
```

| 요소 | 동작 |
|------|------|
| 로고 | 홈(Ticket 목록)으로 이동 |
| VOC 입력 | `/voc/new`로 이동 |
| Ticket 목록 | `/tickets`로 이동 |

### 2.2 상태 Badge

```
┌────────────────────┐
│ ● WAITING_CONFIRM  │
└────────────────────┘
```

| 상태 | 색상 (배경/텍스트) |
|------|-------------------|
| OPEN | #E5E7EB / #374151 |
| ANALYZING | #DBEAFE / #1D4ED8 |
| WAITING_CONFIRM | #FEF3C7 / #D97706 |
| DONE | #D1FAE5 / #059669 |
| MANUAL_REQUIRED | #FEE2E2 / #DC2626 |
| REJECTED | #F3F4F6 / #6B7280 |

### 2.3 긴급도 Badge

```
┌──────┐  ┌────────┐  ┌──────┐
│ low  │  │ medium │  │ high │
└──────┘  └────────┘  └──────┘
```

| 긴급도 | 색상 | 아이콘 |
|--------|------|--------|
| low | #D1FAE5 / #059669 | 🟢 |
| medium | #FEF3C7 / #D97706 | 🟡 |
| high | #FEE2E2 / #DC2626 | 🔴 |

### 2.4 신뢰도 Indicator

```
신뢰도: ████████░░ 85% (높음)
```

| 범위 | 바 색상 | 레이블 |
|------|---------|--------|
| 80%+ | #059669 | 높음 |
| 60~80% | #D97706 | 중간 |
| 60% 미만 | #DC2626 | 낮음 |

### 2.5 Button

```
[ Primary ]  [ Secondary ]  [ Danger ]
```

| 유형 | 배경색 | 텍스트색 | 사용처 |
|------|--------|----------|--------|
| Primary | #2563EB | #FFFFFF | 주요 액션 (제출, 승인) |
| Secondary | #E5E7EB | #374151 | 보조 액션 (취소, 목록) |
| Danger | #DC2626 | #FFFFFF | 위험 액션 (거부) |

### 3.2 컴포넌트 명세

| 컴포넌트 | 타입 | 필수 | 유효성 검사 |
|----------|------|------|-------------|
| 고객명 | TextInput | O | 1~100자 |
| 접수 채널 | RadioGroup | O | email, slack |
| 접수 일시 | DateTimePicker | O | 현재 시점 이전 |
| VOC 내용 | Textarea | O | 1~5000자 |
| 제출 버튼 | Button (Primary) | - | 전체 유효성 통과 시 활성화 |

### 3.3 상호작용

| 이벤트 | 동작 |
|--------|------|
| 제출 클릭 | 로딩 표시 → API 호출 → 성공 시 상세 페이지 이동 |
| 유효성 오류 | 해당 필드 하단에 에러 메시지 표시 |
| API 오류 | 상단 토스트로 에러 메시지 표시 |

| 요소 | 설명 |
|------|------|
| Ticket ID | 클릭 시 상세 페이지 이동 |

| 고객명 | 우측 |
| 생성일시 | 최하단 |

### 4.3 필터 옵션

**상태 필터 (다중 선택)**
```
☑ OPEN
☑ ANALYZING
☑ WAITING_CONFIRM
☐ DONE
☑ MANUAL_REQUIRED
☐ REJECTED
```

**긴급도 필터 (다중 선택)**
```
☐ low
☑ medium
☑ high
```

**정렬**
```
○ 최신순 (기본)
○ 오래된 순
○ 긴급도 높은 순
○ 긴급도 낮은 순
```
